# Image Loading using OpenCV

## Overview
This is a basic Computer Vision (CV) project that demonstrates how to load and display images using OpenCV in Python.

It is part of foundational CV practice for understanding image representation and handling in machine learning pipelines.

---

## Objective
- Load images from local storage
- Understand image data structure
- Display images using OpenCV

---

## Steps

### 1. Import Libraries
- OpenCV (cv2)
- NumPy

### 2. Load Image
- Read image from file path using OpenCV
- Convert image into array format

### 3. Display Image
- Visualize the image using OpenCV display functions

---

## Key Concepts
- Image reading in OpenCV
- Pixel matrix representation
- BGR color format in OpenCV
- Basic image visualization

---

## Tools Used
- Python
- OpenCV
- NumPy

---

## Notes
- Ensure correct file path when loading images
- OpenCV reads images in BGR format by default, not RGB
